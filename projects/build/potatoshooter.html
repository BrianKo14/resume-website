Customizable potato shooter

C# Unity MonoGame Adobe Illustrator

I was putting up with a particularly tedious lecture one day when I was blessed with the particularly great idea of having my friend shoot potatoes at my other friend. I built my first implementation of this game using the MonoGame framework for C#. Later, after I'd discovered the prowess of commercial cross-platform game engines, by means of Unity, I turned weaponizing one's friends to a whole new level.
