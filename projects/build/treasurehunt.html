Treasure Hunt Video Game

Game Design Unity C# Blender

This is a preliminary model for an immersive adventure game set on a deserted island. It's a personal project that I started in 2020, and which quickly outgrew its origins. Built mostly in Unity, the island is 25 km² (9.7 sq mi). A lush, bright green world set on an endless ocean. The player lands on the Upper Pleateau, in a clearing with a view to the Volcano’s Summit, and they are free to roam. Roam through forests, beaches, caves, shipwrecks and ancient temples. The goal? The fancy of every childhood: a treasure hunt—solving puzzles for a way out.
